Los flags vienen del catálogo

Cada elemento tiene configurados en el sistema sus flags: UsaTipo, UsaCantidad, UsaM², UsaMl, UsaEstado. No se pueden cambiar sin una actualización del catálogo.

⚠️

Ningún elemento aplica

Si al abrir una sección ves el aviso naranja "Ningún elemento aplica en esta sección", significa que ese ambiente no tiene esos elementos. Pasa a la siguiente sección.

🔤

Campo Tipo = dropdown

Cuando el elemento usa Tipo, verás un desplegable con las opciones configuradas para ese elemento (MLB, MDR, Simple, Fluorescente…). Si no hay opciones, contacta al administrador.

Botones táctiles grandes

Los botones tienen mínimo 44px de alto para que sean fáciles de tocar desde el celular o tablet. Solo haz tap en el que corresponde — el botón seleccionado se pinta de su color.

💡

Solo un estado a la vez

Al seleccionar REGULAR, los otros dos se desmarcan automáticamente. No puedes dejar el estado vacío si el elemento usa Estado — es un campo requerido antes de guardar.

📋

¿Todos los elementos tienen Estado?

Sí. El campo Estado de Conservación siempre está presente para todos los 27 elementos. Es el único campo universal del formulario.

🏗️

Estructuras — solo Estado

Para los 5 elementos de Estructuras (Columnas, Vigas, Muro Portante, Losa Aligerada, Losa Maciza), el único campo que aparece es el Estado B/R/M. No hay Tipo, Cantidad ni Medidas.

Slide 06

Guardar por slot (elemento a elemento)

No hay un botón "Guardar todo". Cada elemento se guarda de forma independiente. Esto protege tu trabajo en caso de pérdida de señal.

1
Llenas los campos del elemento
Seleccionas el Tipo (si aplica), ingresas Cantidad o Medidas (si aplican) y eliges el Estado B/R/M.
2
Haces click en Guardar
Solo se envía ese elemento al servidor. No se afectan los demás. Mientras guarda aparece un spinner en el botón.
3
El slot queda guardado
La card muestra el badge verde "✓ Guardado". Los valores quedan visibles como solo-lectura. El contador del área sube en 1.

📶

Protección ante pérdida de señal

Como se guarda slot por slot, si la conexión se corta a mitad de trabajo, solo pierdes el elemento que estabas guardando en ese momento. Todo lo anterior ya está en el servidor.

🔄

Upsert — crear o actualizar

Si el elemento ya existía (badge verde), al hacer click en Guardar se actualiza el registro. No se crea uno nuevo. El sistema siempre usa la regla "uno por elemento por ambiente".

⚡

El progreso sube en tiempo real

Tanto el contador del área (ej. "4 / 5") como el Progreso Total del panel lateral se actualizan inmediatamente después de cada guardado exitoso.

Slide 07

Auto-avance y botón Limpiar

Dos comportamientos inteligentes que aceleran el trabajo: el sistema avanza solo al terminar un área, y te permite borrar un elemento si cometiste un error.

⚡ Auto-avance automático
✅
Guardas el último elemento del área
Al guardar el elemento que faltaba, el contador del área llega a su total (ej. 5/5). El badge cambia de ámbar a verde.
🎯
La sección se cierra automáticamente
El accordion colapsa la sección que acabas de completar. No necesitas hacer click en ningún lado.
👇
La siguiente sección incompleta se abre
El sistema detecta cuál es la próxima área que tiene elementos pendientes y la expande automáticamente. Puedes continuar de inmediato.
🏁
Al completar todo el inventario
Cuando todos los elementos de todas las secciones están guardados, el Progreso Total llega al máximo y el estado del ambiente cambia a Completado en la lista.
🗑 Botón Limpiar (DELETE físico)
🗑
Limpiar elemento
Esta acción no se puede deshacer

¿Eliminar el registro de Interruptor?
El elemento quedará en blanco y podrás registrarlo de nuevo.

⚠️

¿Cuándo usar Limpiar?

Cuando cometiste un error en un elemento ya guardado: tipo incorrecto, cantidad equivocada, estado mal seleccionado. Al limpiar, el slot desaparece de la base de datos — el contador del área baja en 1 y el elemento queda listo para volver a registrarse.

💡

El botón Limpiar solo aparece si ya hay datos

Si el elemento no ha sido guardado nunca, el botón Limpiar está oculto. Solo aparece cuando la card tiene el badge verde "✓ Guardado".
